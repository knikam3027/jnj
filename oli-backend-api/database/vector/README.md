# Vector Database Configuration

This directory contains configuration and setup for vector database integration.

## Purpose
- Store and retrieve vector embeddings for semantic search
- Enable AI-powered chat context and memory
- Support RAG (Retrieval Augmented Generation) workflows

## Supported Vector Databases
- Pinecone
- Weaviate
- Qdrant
- ChromaDB
- pgvector (PostgreSQL extension)
